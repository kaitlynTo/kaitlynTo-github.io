RESPONSIVE WEB DESIGN

"Responsive web design (RWD) is an approach to web design that makes web pages render well on a variety of devices and window or screen sizes."(Wikipedia,2020).

In other words, responsive web design is to make a web page look nice on all possible devices so that it increases user experience.

To make a web page responsive, you may design in a way you want, or you might use existing CSS frameworks, such as, bootstrap and w3.css and so on.

First of all, we set the viewport to enable a web page responsive within the meta tag. This will ensure the content's width to be relative to the device's width.

Using media queries might be the most crucial part of designing responsive web. By using '@media' keyword, we can design differently for different screen sizes. For example, if we want to show different design when the screen size is smaller than 500px, we add a breakpoint at 500px. (e.g. @media screen and (max-width: 500px){...}) Everything inside the curly brackets will be adapted only when the screen is smaller than 500px.

Besides, although I realized after I already created my web page, it is better to design for mobile screen first because it will make the page load faster on smaller devices with fewer elements and it is easier to enhance from the essential elements rather than the other way around.

For my web page, I used a media query so that the display will be changed when the screen is smaller than 780px. Also, I used the ‘Boostrap’ framework to organize my contents in a 3-equal-column layout using the CSS of the class "col-sm-4" which is already defined in the bootstrap. In the way, it significantly reduced my time to design CSS by myself.
